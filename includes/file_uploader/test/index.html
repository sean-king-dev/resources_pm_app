jQuery Multiple File Upload Plugin - Test Suite

This is the template you can use to test your integration of this plugin.

The form will be submitted to a file called test.asp which will report the files received using Jacob Gilley's Pure ASP File Upload - but the files will never be saved on the server.

Because of the way this upload script works, I've configured the plugin to give each file input element a unique name like this: 

<input type="file" name="files[]" class="multi {namePattern:'$name$i'}" maxlength="5"/>

File Selection



Clear Selection

Use this code to reset selected files:
$('input:file').MultiFile('reset');
(click on it to execute it)

Ajax Upload Example

This example has been configured to automatically reset the file selection when the response from the server is received. 

// SAMPLE CODE
$('#ajax-button').click(function(){
  $(this.form).ajaxSubmit({
    success: function(r){
      $('#ajax-results').html(r);
      $('input:file').MultiFile('reset');
    }
  });
});

Native Upload Example

This example has not been configured to reset the file selection automatically, but you can achieve this by executing this code when the response is received: 

$('input:file').MultiFile('reset');

PS.: Of course, you'd only need to do this if you used an iframe to submit the form because conventional submission would take the users away from this page anyway.
